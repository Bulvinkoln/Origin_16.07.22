Для работы с удаленными репозиториями необходимо:
1. сделать fork (отвлетвление) того репозитория, с которым собираемся работать
2. сделать git clone + ссылка на СВОЙ репозиторий
3. сделать новую ветку и в ней сделать изменения
5. зафиксировать изменения коммитами
6. отправить свою версию на github
на сайте github нажать pull request, что бы предложить свои изменения
